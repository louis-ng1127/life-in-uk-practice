# Life in the UK Practice

A free, mobile-friendly practice app for preparing for the Life in the UK Test.

[**Open the live app →**](https://life-in-uk-practice.pages.dev)

If this project helps you, please give it a **Star** on GitHub. It helps more learners discover the project.

## Features

- 435 practice questions from 18 practice exams
- Choose a question range or practise all questions
- Randomised question order on every session
- Randomised answer order with correct-answer mapping preserved
- Single-choice and multiple-choice questions
- Reveal the answer and explanation only when requested
- Local progress, score and incorrect-question tracking
- Responsive layout for iPhone, iPad, Mac, Windows and Android
- Installable Progressive Web App (PWA) with offline support
- No account required; learning records stay in the current browser

## Use it on iPhone or iPad

1. Open the [live app](https://life-in-uk-practice.pages.dev) in Safari.
2. Tap **Share**.
3. Choose **Add to Home Screen**.
4. After the first successful load, cached questions can be used offline.

## Privacy

The app does not require registration. Practice progress is stored locally in the browser using local storage and is not uploaded by the app.

## Run locally

Serve the repository with any static web server. For example:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Technology

Vanilla HTML, CSS and JavaScript, deployed on Cloudflare Pages. No framework or build step is required.
